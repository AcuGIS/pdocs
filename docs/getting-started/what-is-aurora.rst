What is Aurora
==============

Aurora is a geospatial platform for ingesting, managing, analyzing, and publishing spatial data and applications.

It supports vector and raster data, maps, dashboards, pipelines, field data synchronization, and reporting within a unified environment.

Key capabilities include:

- Spatial data ingestion from GIS and enterprise sources
- Interactive map publishing
- Dashboard and analytics creation
- Field data synchronization (GeoSync)
- Automated data pipelines
- Reporting and visualization apps
