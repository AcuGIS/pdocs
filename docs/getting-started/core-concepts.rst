Core Concepts
=============

Aurora organizes spatial content into several core components.

Datasets
--------

Spatial data stored in Aurora.

Maps
----

Interactive visualizations of datasets.

Dashboards
----------

Analytical views combining maps and charts.

Pipelines
---------

Automated data ingestion and transformation workflows.

GeoSync
-------

Field data synchronization with mobile platforms.

Apps
----

Published applications such as reports, timelines, and presentations.
